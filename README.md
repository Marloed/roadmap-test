# Python Roadmap Backend API 

Учебный проект на Python с использование BlackSheep, PostgreSQL и Piccolo ORM.

Проект для изучения:
- HTTP-запросов и ответов
- JSON API
- роутинга
- CRUD
- PostgreSQL
- ORM
- миграций
- HTTP-статусов
- базовой структуры backend-проекта

## Стек

- Python 3.12
- BlackSheep
- PostgreSQL
- Piccolo ORM
- Uvicorn
- python-dotenv
- Git
